Octopress Deploy to Github Pages

1. Install Ruby on Ubuntu

tar -zxf ruby-2.5.0.tar.gz

cd ruby-2.5.0

./configure –prefix=/usr/local/ruby

make

sudo make install

sudo gedit /etc/profile

export RUBY_HOME =/usr/local/software/ruby-2.5.0

export PATH=${RUBY_HOME}/bin:$PATH

source /etc/profile

ruby -v

2. Install Octopress

check the git

git –version

git clone octopress

git clone git://github.com/imathis/octopress.git octopress

install others

cd octopress gem install bundler rbenv rehash bundle install rake install

3. Octopress deploy to Github Pages

  1. rake setup_github_pages

  2. rake generate

  3. rake deploy

  4. github

    git add .
    git commit -m ‘initial source commit’
    git push origin source
